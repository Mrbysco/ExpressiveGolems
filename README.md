# Expressive Golems #

## About ##
**Expressive Golems** adds more faces that show when Snow Golems have their pumpkin removed by shears. The mod adds 10 different faces that a Snow Golem can have when they are no longer wearing a Pumpkin on their head.

The mod comes with the following faces:

-   Default - The default vanilla face
-   Creeper - A creeper inspired face
-   Good Times With - A Good Times With Scar inspired face
-   Happy - A happy ghast inspired face
-   Panda - A panda inspired face
-   Slime - A slime inspired face
-   Spider - A spider inspired face
-   Strider - A strider inspired face
-   Undead - A zombie inspired face
-   Random 1, 2 and 3 - Random non-mob inspired faces
